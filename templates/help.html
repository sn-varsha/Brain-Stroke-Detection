File Requirements

Supported Formats
  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • DICOM (.dcm)
  • TIFF (.tiff, .tif)
Image Specifications
  • Minimum resolution: 256x256 pixels
  • Optimal resolution: 512x512 pixels
  • Maximum file size: 50 MB
  • Recommended: High contrast images
Image Content
  • Brain CT scan images only
  • Axial view preferred
  • Clear visibility of brain structures
  • Minimal artifacts or noise
Technical Requirements
  • Modern web browser
  • Stable internet connection
  • JavaScript enabled
  • Updated browser version

Important Medical Disclaimer

This system is for research and educational purposes only. The results provided by NeuroDetect are not intended to replace professional medical diagnosis, treatment, or consultation. Always consult with qualified medical professionals for any health concerns or medical decisions. The accuracy of the system may vary, and false positives or false negatives are possible.

Tips for Best Results

  • Use high-quality, high-resolution CT scan images for better accuracy
  • Ensure the image is properly oriented (axial view recommended)
  • Upload images with good contrast between brain tissue and background
  • Avoid images with significant motion artifacts or noise
  • Make sure the entire brain is visible in the scan
  • Use images that haven't been heavily compressed or modified
  • For best results, use images from recent CT scans (within 24-48 hours)

Troubleshooting

Upload Failed

Check that your file is in a supported format (JPEG, PNG, DICOM, TIFF) and under 50MB. Ensure you have a stable internet connection.

Analysis Taking Too Long

Large files may take longer to process. If analysis takes more than 30 seconds, try refreshing the page and uploading a smaller file.

No Preview Showing

Ensure your file is a valid image format. Try converting your image to JPEG or PNG format if you're having issues.

Error Messages

If you receive error messages, check your internet connection and try uploading again. For persistent issues, contact technical support.

Browser Compatibility

Make sure you're using an updated version of Chrome, Firefox, Safari, or Edge. Clear your browser cache if you experience issues.
